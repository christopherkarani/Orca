# Aegis Edge Customer Pilot Package

This package helps a drone or autonomy company evaluate Aegis Edge as a simulation/SITL/bench-preparation safety-policy runtime. It is customer-evaluation material only: no real flight, no live aircraft control, no certification claim, no regulatory approval claim, no detect-and-avoid claim, and no autopilot replacement claim.

## Package Shape

The pilot materials are local files. They do not require external network access, hosted telemetry, real secrets, real customer names, or real drone hardware.

Use the templates in [templates](templates/) for draft commercial and security-review workflows. Legal-style templates are drafts only and require legal review before use.

Use the examples in [examples](examples/) as deterministic sample outputs with fake adapter evidence. They are not real flight evidence.
