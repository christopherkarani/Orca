# Health Red-Team

Health red-team fixtures exercise stale agent heartbeat, stale adapter heartbeat, stale MAVLink heartbeat, stale telemetry, audit failure, missing policy, missing home position for RTH, critical battery emergency LAND policy, event queue depth, and fake-secret redaction.

Fixtures are deterministic and local. They do not require real hardware, external network, hosted telemetry, or real-flight operation. Skipped, unsupported, and inconclusive results do not count as passed.

Health red-team evidence is not real-flight readiness, not an autopilot replacement, not detect-and-avoid, and not regulatory certification.
