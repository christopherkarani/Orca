# Integration Readiness Checklist

- [ ] Fake adapter evaluation possible.
- [ ] PX4 SITL available or explicitly out of scope.
- [ ] ArduPilot SITL available or explicitly out of scope.
- [ ] MAVLink endpoints identified with placeholders only.
- [ ] Command surface identified.
- [ ] Policy scope defined.
- [ ] Scenario list defined.
- [ ] No real hardware required.
- [ ] No real secrets required.
- [ ] Audit output path agreed.
- [ ] Red-team fixtures selected.
- [ ] Success criteria agreed.
- [ ] Safety boundary agreed.

No checklist item authorizes real flight, live aircraft control, or external network transfer.
