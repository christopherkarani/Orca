# FAQ

## Is Aegis Edge a flight controller?

No. It is a policy, mediation, audit, red-team, and evidence runtime for simulation/SITL/bench-preparation evaluation.

## Does it replace PX4 or ArduPilot?

No. It does not replace autopilot failsafes or flight-control behavior.

## Does it require cloud connectivity?

No. Normal evaluation is local and offline.

## Does it send telemetry externally?

No hosted telemetry is required. Data guard examples evaluate local payload and endpoint fixtures.

## Does the pilot require real secrets?

No. Use placeholders only.

## Is this a certification package?

No. It is a non-certification customer-evaluation package.

## Can this evaluate real flight?

No. Real flight and live aircraft control are out of scope.
