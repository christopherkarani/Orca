# Aegis Edge v1.1.0 Release Notes

Aegis Edge v1.1.0 packages are prepared for simulation/SITL/customer-evaluation and bench-preparation only. This is not real-flight readiness.

Included release surfaces:

- MAVLink gateway.
- PX4 SITL and ArduPilot SITL opt-in local simulation support.
- Safety enforcement, operator approvals, emergency modes, data guard, runtime health, watchdog diagnostics, audit/replay, safety-case reports, red-team/fault injection, deployment/ARM64 package metadata, customer proof docs, and customer pilot package.

Known limitations:

- Not real-flight readiness.
- Not certification or regulatory approval.
- Not detect-and-avoid.
- Not autopilot replacement.
- Not complete MAVLink coverage.
- Does not promise every unsafe action is prevented.

Verify artifacts with `checksums.txt` before install.
