# What Edge Does Not Prove

Edge does not prove the aircraft is safe for flight.

It also does not prove:

- It does not prove compliance with FAA/EASA/CAA rules.
- BVLOS authorization.
- It does not prove detect-and-avoid capability.
- Airworthiness.
- Autopilot correctness.
- Hardware reliability.
- Real-world flight safety.
- Operator training adequacy.
- Physical kill-switch behavior.
- Radio-link reliability.
- It does not prove all MAVLink commands are covered.
- All vehicle modes are covered.
- All customer-specific integrations are safe.

Edge is not a flight controller, not an autopilot replacement, and not regulatory approval.
