# Claims To Avoid

Do not use these claims except in negative context as examples of what Edge does not provide.

- "certified safe"
- "guarantees safety"
- "FAA approved"
- "BVLOS-ready"
- "flight-ready"
- "real-flight-ready"
- "replaces autopilot"
- "autopilot replacement"
- "detect-and-avoid"
- "prevents all unsafe commands"
- "covers all MAVLink commands"
- "regulatory approval"
- "airworthiness approval"
- "complete safety system"
- "production aircraft validation"

## Safer Replacements

- Use "evaluation evidence" instead of approval language.
- Use "supported scenarios" instead of universal coverage.
- Use "policy and audit layer" instead of aircraft-control language.
- Use "simulation/SITL/bench-preparation" instead of field-deployment wording.
