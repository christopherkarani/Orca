# Examples

These examples are local-only and do not require real secrets, real LLMs, or external network services.

- `leaky-agent-demo/`: deterministic scary demo.
- `policies/`: launch-ready policy examples.
- `effect-packs/`: user effect-pack YAML (classification extensions for `effects:`).
- `mcp/`: stdio MCP proxy examples.
- `ci/`: CI snippets.
- `staged-writes/`: staged write review notes.
- `network/`: network policy examples.
- `commands/`: command guard examples.
- `edge/`: fake/simulation-only Edge policy evaluation examples.
